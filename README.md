# Bazar-PIQ — Archivos para instalar como App

Con estos archivos, Bazar-PIQ se instala en Android e iPhone
igual que WhatsApp o Facebook — aparece en el menú de aplicaciones.

## PASO 1 — Crear cuenta en GitHub (gratis)
1. Ve a https://github.com
2. Regístrate con el nombre de usuario: quinielas580
3. Confirma tu correo

## PASO 2 — Crear el repositorio
1. Click en "New"
2. Repository name: bazarpiq-pwa
3. Marca "Public" y "Add a README file"
4. Click "Create repository"

## PASO 3 — Subir los archivos
Sube estos archivos al repositorio:
- manifest.json
- sw.js
- icon-192.png
- icon-512.png
- index.html (el pequeño)
- logo.png

## PASO 4 — Activar GitHub Pages
1. Settings → Pages
2. Branch: main, carpeta: / (root)
3. Save → espera 2-3 minutos
4. URL: https://quinielas580.github.io/bazarpiq-pwa/

## PASO 5 — Actualizar Index.html en Apps Script
Reemplaza el contenido de Index.html en tu proyecto de Apps Script
con el nuevo Index.html incluido en estos archivos.
Luego: Implementar → Administrar implementaciones → Nueva versión.
